# 04 데이터 링크 계층

- Data link layer is concerned with direct link networks
- Two nodes directly connected by a single point-to-point link
- A broadcast link segment with multiple attachments



## Node Configuration

- Network adapter connects to I/O bus & network link
  - Device driver manages a network adapter
  - Manage message passes between main memory and network adapter buffer



## Data Link Layer

- Functions
  - framing
  - Error control
  - Flow control
  - Medium access control (MAC) - LAN
- Standard protocols
  - HDLC (High-level data Link Control) - ISO
  - LAPB (Link Access Protocol Balanced) - ITU
  - LLC (Logical Link Control) - IEEE
    - Type 1, 2, 3
